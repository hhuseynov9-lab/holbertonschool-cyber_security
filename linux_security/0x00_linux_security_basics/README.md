# Linux Security - Protocols and Servers

This repository is dedicated to the study and implementation of Linux security concepts, specifically focusing on network protocols, firewall configurations, and server hardening.

## Technologies & Tools Used
* **OS:** Kali Linux
* **Shell:** Bash
* **Firewall:** iptables
* **Version Control:** Git / GitHub

## Learning Objectives
* Understanding network protocols and server security.
* Configuring firewall rules using `iptables` to secure network traffic.
* Writing automation scripts for Linux system administration and security hardening.

## Requirements
* All scripts are written and tested on Kali Linux.
* Scripts are executable and comply with the `#!/bin/bash` standard.
