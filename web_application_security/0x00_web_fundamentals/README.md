# 0x00 Web Fundamentals

## Description

This project introduces the fundamentals of web application security. It covers the basic concepts required before performing web application penetration testing.

During this project, we:

* Configure the local `hosts` file to resolve a custom domain.
* Connect to the target machine.
* Test connectivity using `curl`.
* Understand HTTP redirection.
* Access the target web application through a browser.

## Learning Objectives

* Understand the purpose of the `/etc/hosts` file.
* Learn how domain names are resolved locally.
* Use `curl` to send HTTP requests.
* Understand HTTP redirects (301/302).
* Prepare the environment for web application security testing.

## Requirements

* Kali Linux
* OpenVPN or Sandbox access
* Firefox
* Terminal
* curl
